FAQ

1. Who is this course for?

This course is designed for anyone who is interested in learning JavaScript, whether you're a beginner or have some experience.

2. What tools do I need to follow the course?

You only need a code editor and a browser to start following the course.

3. Is this a JavaScript course?

Yes, this course focuses primarily on JavaScript and how to use it effectively.

4. Why is this course free?

We believe in making learning accessible to everyone, so we offer this course for free.

5. Are the lessons listed on this course?

Yes, you can find all the lessons on the course page, listed in order of learning.

6. I spotted a bug, what can I do?

If you find a bug, please report it via the contact form on the course page.

7. Can I use frameworks instead of vanilla JavaScript?

While frameworks like React or Vue.js are popular, learning vanilla JavaScript will give you a strong foundation before moving on to frameworks.

8. How can I contact you if I have questions?

You can submit your questions through the contact section on the website, or reach out via email.
